  * ans 05 :: C, D
  *********--*************************************************************************************
  *	
  *  ====------------=----------------------------------------------------------------------------
  *       A. replace using import lib.*;          
  *  ====------------=----------------------------------------------------------------------------
  *       PS         : import lib.* can not import package lib.tool together
  *                  : import lib.* 无法一起导入软件包 lib.tool
  *      ------------=----------------------------------------------------------------------------
  *  
  *  	
  *  ====------------=----------------------------------------------------------------------------
  *       B. replace using import lib.*.Tool;     
  *  ====------------=----------------------------------------------------------------------------
  *       PS         : incorrect import syntax
  *                  : 错误的导入语法
  *      ------------=----------------------------------------------------------------------------
  *  	
  *  
  *  ====------------=----------------------------------------------------------------------------
  *  [V]  C. replace using import lib.tool.Tool;  
  *  ====------------=----------------------------------------------------------------------------
  *       PS         : correct import syntax ( import only one class lib.tool.Tool )
  *      ------------=----------------------------------------------------------------------------
  *  
  *  
  *  ====------------=----------------------------------------------------------------------------
  *  [V]  D. replace using import lib.tool.*;     
  *  ====------------=----------------------------------------------------------------------------
  *       ps         : correct import syntax ( import all class in package lib.tool )
  *      ------------=----------------------------------------------------------------------------
  *  
  *  
  *  ====------------=----------------------------------------------------------------------------
  *       E. replace using import lib.tool.Tool.*;
  *  ====------------=----------------------------------------------------------------------------
  *       ps         : incorrect import syntax, if you want to use static import 
  *                    you can use follow syntax [ import static lib.tool.Tool.*; ]
  *                  : 导入语法不正确, 如果要使用静态导入，您可以使用以下语法
  *                    [ import static lib.tool.Tool。*; ]
  *      ------------=----------------------------------------------------------------------------
  *  
  *  
  *  ====------------=----------------------------------------------------------------------------
  *       F. none of these changes will let code compile successful.
  *  ====------------=----------------------------------------------------------------------------
